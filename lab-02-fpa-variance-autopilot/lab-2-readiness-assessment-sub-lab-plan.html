Split Lab 2 into focused, self-contained sub-labs that each have a single deliverable. Participants can stop at any sub-lab.

Lab 2.1 Add MCP Server & PA Tools to Orchestrate 90% ready

Goal: Register the Planning Analytics MCP server in watsonx Orchestrate and verify all 9 PA tools are discoverable.

  • Navigate to Integrations → Add MCP Server
  • Enter URL, transport (streamable-http), Basic Auth
  • Verify all 9 PA tools are listed
  • Run a manual tool call from the Tools panel
Deliverable: MCP integration active in Orchestrate
Assets needed: README ex2.1–2.3 Minor polish needed
Lab 2.2 Import SalesLens REST API as CRM & ERP Tools 85% ready

Goal: Import the SalesLens OpenAPI spec into Orchestrate to create 6 CRM/ERP tools, and validate with a live call.

  • Start SalesLens app (local or Code Engine URL)
  • Integrations → Add REST API → paste /api-spec URL
  • Set X-Api-Key credential
  • Confirm 6 tools created (getCrmVarianceContext, getErpCostContext …)
  • Test getCrmVarianceContext manually in Try-it panel
Deliverable: saleslens tools active in Orchestrate
Assets needed: sample-app/ (running) openapi.json Both complete
